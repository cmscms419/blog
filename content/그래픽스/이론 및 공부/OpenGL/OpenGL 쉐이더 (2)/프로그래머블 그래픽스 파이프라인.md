### 하드웨어 발전

그래픽스 전용 CPU → GPU 

shader = small, special-purpose program

- shader program = GPU에서 돌아가는 samll-size 프로그램, 병렬 처리
- shader language = 프로그래밍 언어
- shader compiler = 전용 컴파일

### programmable Pipeline

현재는 일부를 shader로 구현 → programmable pipeline

vertex shader : vertex processing VLSI를 shader로 대체

fragment shader : fragment processing VLSI를 shader로 대체

대부분 쉐이더 프로그램을 지정합니다.
