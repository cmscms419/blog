### vertex input

꼭지점 입력

### vertex precessing

사용자의 vertex data를 좌표 변환

NDC(normalized device coordinate)

### primitive assembly

vertex 결합 → graphics primitive

1 vertex → 점 point

2 vertex → 선분 a line segment

3 vertex → 삼각형 a triangle

### rasterization

raster :  텔레비전 수상기에서, 영상 신호가 없을 때 나타나는 희게 빛나는 화면.

primitive에 포함되는 pixel만 선정하는 

### fragment processing

fragment = pixel(location) + associated data

각각의 fragment color를 선택합니다.

### blend

최종 결과물에 다가 마지막 효과를 준다.

fragment 단위 처리로 다양한 효과 → 전체 이미지에 뿌리게 된다.

post-processing(후처리) 단계

### framebuffer

결과물이 framebuffer에 저장되면서, 화면에 뿌려진다.
