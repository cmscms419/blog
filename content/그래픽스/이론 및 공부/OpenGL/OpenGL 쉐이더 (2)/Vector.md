# Vector

### Scalar

실수 값 1개

알파 베타 감마 

### Position

위치 : 좌표 coordinate value

point 라고도 함

p, q, r

### Vector

변화량(방향 + 크기)

u, v, w

가능한 연산

vector = vector + vector(vector의 합성)

### **position vs vector**

포지션은 위치

position = origin + vector

p = o + u

벡터는 이동양

### unit vector 단위 벡터

magnitude 크기

벡터의 크기

vector normalization

벡터의 크기를 1로 만든다.

### 벡터 공간

스칼라와 벡터가 정의되어 있는 공간

- 스칼라-스칼라 연산
- 스칼라-벡터 연산
- 벡터-벡터 더하기, 곱하기
- basis vectors 좌표축 상의 유닉 벡터

### 어파인 공간

벡터 공간 + 포지션

- 스칼라-스칼라 연산
- 스칼라-벡터 연산
- 벡터-벡터 더하기, 곱하기
- 포지션-벡터 더하기
- 포지션-포지션 빼기
- 좌표 = 원점 + basis vector

### 유클리드 공간

- 어파인 공간에서 “거리”가 정의되어 있는 공간
- 원점에서 좌표까지의 거리
