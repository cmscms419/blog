## GroupShared

![Untitled](attachments/Untitled%201.png)

그룹들 끼리는 공유할 수 없다.

하지만 한 그룹안에 여러개의 쓰레드들은 공유할 수 있다.

GroupSharedMemory를 공유할 수 있다. 일종의 캐시 메모리이다.

다만, GroupSharedMemory의 메모리 크기는 16KB이다.

쓰기는 256byte만 쓸 수 있다.

- 가로
    - tgx
    - 쓰래드들이 가로 방향으로 메모리 공유
- 세로
    - tgy
    - 쓰래드들이 세로 방향으로 메모리 공유

Dispatch 할 때도 Group 캐쉬를 염두해 두었다.

Shader

```cpp
static const int blurRadius = 5;

#define N 256
#define CACHE_SIZE (N + 2*blurRadius)
```

한 그룹들은 같이 사용하는 캐쉬 메모리들은 빠르게 접근해서 쓴다.

그리고 캐쉬가 정리가 되고 난 다음에

GroupMemoryBarrierWithGroupSync() → 모든 쓰레드가 완료될 때까지 기다린다.

```cpp
[unroll]
for (int i = -blurRadius; i <= blurRadius; ++i)
{
    int k = gtID.x + blurRadius + i;
    blurColor += weights[i + blurRadius] * groupCache[k];
}

outputTex[dtID.xy] = blurColor;
```

캐쉬로 부터 값을 읽어와서, 일을 시작합니다.
