# 영어 단어

# Prerequisites

pre + requisite

앞 + 요구하다 = 전제조건

## ex)

> One of the key prerequisites for success is patience.
> 
> 
> 성공을 위한 가장 중요한 전제조건 중 하나는 인내야.’

# Functionality

function : 기능, 함수(명사) 작동하다(동사)

functional : 기능적인 (형용사)

functional + ly(부사)로 만들어진 단어이다.

기능적으로(부사)

## ex)

> new software with additional functionality.
> 
> 
> 부가 기능이 추가된 새 소프트웨어
