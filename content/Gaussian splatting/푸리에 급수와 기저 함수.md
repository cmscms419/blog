# 푸리에 급수(Fourier Series) 정리 - 3D Gaussian Splatting을 위한 기초
---
## 📚 학습 목표

3D Gaussian Splatting(3DGS)에서 **구면 조화 함수(Spherical Harmonics, SH)**를 이해하기 위한 기초 단계입니다.
### 왜 배워야 하나?

3DGS에서는 각 가우시안이 **보는 방향에 따라 다른 색상**을 표현해야 합니다:
- 예: 금속 표면은 보는 각도에 따라 반사색이 달라짐
- 이를 **view-dependent color**라고 함

**학습 순서:**
1. **푸리에 급수** (이 노트북) → 1D 신호를 사인/코사인으로 분해
2. **2D 푸리에 변환** → 2D 주파수 분석
3. **구면 조화 함수** → 구 표면의 기저 함수 (최종 목표)

**실용적 이점:**
- 메모리 효율: 무한한 방향의 색상 정보를 소수의 SH 계수(보통 9~16개)로 압축
- 계산 효율: 렌더링 시 빠른 색상 계산 가능
- 부드러운 보간: 방향 변화에 따른 자연스러운 색상 전환

푸리에 급수는 이 모든 것의 출발점입니다. "복잡한 것을 간단한 기저 함수들의 조합으로 표현한다"는 핵심 아이디어를 먼저 1차원에서 확실히 이해하고 나면, 이를 3차원 구면으로 확장하는 것이 훨씬 수월해집니다.

## 🎯 푸리에 급수란?

> **"어떤 복잡한 주기 신호도 단순한 사인/코사인 파동들의 합으로 표현할 수 있다"**

수학적으로:

$$
f(x) = a_0 + \sum_{n=1}^{\infty} \left[ a_n \cos(nx) + b_n \sin(nx) \right]
$$

## 🔬 왜 푸리에 급수를 사용하는가?

### 1️⃣ **데이터 압축**

| 원본 데이터 | 푸리에 표현 |
|------------|------------|
| 1000개 샘플점 | 몇 개의 계수 $(a_n, b_n)$ |

복잡한 신호를 **소수의 숫자**로 표현 가능!

### 2️⃣ **신호 분해 (Analysis)**

```
복잡한 신호 → [푸리에 분석] → 어떤 주파수가 얼마나 포함되어 있는지
```

예: 음악 → 어떤 음(주파수)이 포함되어 있는지 분석

### 3️⃣ **신호 합성 (Synthesis)**

```
원하는 주파수들 → [합성] → 원하는 신호 생성
```

### 4️⃣ **근사 (Approximation)**

적은 항으로도 원본 신호의 "대략적인 모양"을 표현:

| 항 개수 | 표현력 |
|--------|--------|
| 1개 | 기본적인 진동만 |
| 3개 | 대략적인 형태 |
| 10개 | 꽤 정확한 근사 |
| ∞개 | 완벽한 재현 |

---

## 🧮 수학적 의미

### 직교성 (Orthogonality)

사인/코사인 함수들은 서로 **직교**합니다:

$$
\int_0^{2\pi} \sin(mx) \cdot \sin(nx) \, dx = \begin{cases} 0 & (m \neq n) \\ \pi & (m = n) \end{cases}
$$

이것이 의미하는 바:
- 각 기저 함수는 **독립적인 정보**를 담음
- RGB 색상의 R, G, B가 서로 독립인 것과 같음
- 계수를 **개별적으로** 추출 가능

### 계수의 의미

| 계수 | 의미 |
|------|------|
| $a_0$ | 평균값 (DC 성분) |
| $a_n$ | n번째 코사인 성분의 강도 |
| $b_n$ | n번째 사인 성분의 강도 |

---

## 🎯 3D Gaussian Splatting과의 연결

| 푸리에 급수 (1D) | 구면 조화 함수 (3D) |
|------------------|---------------------|
| 각도 θ (0~360°) | 방향 (θ, φ) - 구면 전체 |
| sin(nθ), cos(nθ) | $Y_l^m(\theta, \phi)$ |
| 계수 $a_n, b_n$ | SH 계수 (9~16개) |
| 1D 신호 표현 | **방향별 색상** 표현 |

```
푸리에: f(θ) = Σ cₙ · basis_n(θ)
    ↓ 3D 구면으로 확장
SH:    color(θ,φ) = Σ cₗₘ · Yₗᵐ(θ,φ)
```

## 📝 핵심 요약

```
┌─────────────────────────────────────────────────────────┐
│  복잡한 신호 = 단순한 기저 함수들의 가중합               │
│                                                         │
│  f(x) = c₁·basis₁(x) + c₂·basis₂(x) + c₃·basis₃(x) + ... │
│         ────────────   ────────────   ────────────       │
│            항 1            항 2           항 3           │
│                                                         │
│  • 기저 함수: sin, cos (서로 직교)                       │
│  • 계수: 각 기저의 "양"을 결정                          │
│  • 더 많은 항 = 더 정확한 근사                          │
└─────────────────────────────────────────────────────────┘
```

슬라이더를 직접 움직여서 **Sum (초록/마젠타 선)**이 어떻게 변하는지 관찰하면, 푸리에 급수의 본질을 체감할 수 있습니다!

## 1. 기저 함수(Basis Functions) 개념

### 비유로 이해하기

| 비유        | 설명                           |
| --------- | ---------------------------- |
| **RGB 색** | 빨강, 녹색, 파랑을 섞어 모든 색을 표현      |
| **합창단**   | 소프라노, 알토, 테너, 베이스가 합쳐져 화음 생성 |

### 핵심 원리

$$
\text{복잡한 신호} = \sum_i (\text{계수}_i \times \text{기저 함수}_i)
$$

- **기저 함수**: $\sin(t), \sin(2t), \sin(3t), \cos(t), \cos(2t), ...$
- **계수(Coefficient)**: 각 기저 함수의 "양"을 결정

> 💡 **핵심 통찰**: 무한히 많은 기저 함수가 필요 없음! 적절히 선택된 **몇 개 항**만으로도 신호의 본질적 특징을 포착 가능

---

## 2. 삼각함수의 두 가지 표현

사인과 코사인은 하나의 파동으로 결합 가능:

$$
A_n \cos(\omega t + \phi_n) = a_n \cos(\omega t) + b_n \sin(\omega t)
$$

| 용어 | 공식 | 의미 |
|------|------|------|
| **진폭(Amplitude)** | $A_n = \sqrt{a_n^2 + b_n^2}$ | 파동의 세기 |
| **위상(Phase)** | $\phi_n = \arctan\left(\frac{-b_n}{a_n}\right)$ | 파동의 시작점 이동 |

---

## 3. 중첩(Superposition)

서로 다른 주파수의 기저 함수들을 **가중치를 곱해 더함**:

$$
\text{신호} = c_1 \sin(x) + c_2 \sin(2x) + c_3 \sin(3x) + \ldots
$$

### 예시: 사각파(Step Function) 근사

| 계수    | 값    | 역할     |
| ----- | ---- | ------ |
| $c_1$ | 1.91 | 기본 주파수 |
| $c_3$ | 0.64 | 3차 고조파 |
| $c_5$ | 0.38 | 5차 고조파 |
| $c_7$ | 0.27 | 7차 고조파 |
| $c_9$ | 0.21 | 9차 고조파 |

→ 더 많은 항을 추가할수록 원본 신호에 가까워짐

---

## 4. 이산 푸리에 분석 (계수 추출)

샘플링된 데이터에서 푸리에 계수를 계산하는 공식:

$$
a_n = \frac{2}{N} \sum_{k=0}^{N-1} f_k \cos\left(\frac{2\pi n k}{T}\right)
$$

$$
b_n = \frac{2}{N} \sum_{k=0}^{N-1} f_k \sin\left(\frac{2\pi n k}{T}\right)
$$

| 변수    | 의미               |
| ----- | ---------------- |
| $N$   | 데이터 포인트 수        |
| $T$   | 전체 주기            |
| $f_k$ | k번째 샘플 값         |
| $n$   | 고조파 인덱스 (주파수 결정) |

---

## 5. 실제 예시: 서울 온도 데이터

3년간 일일 온도 데이터를 푸리에 급수로 근사:

| 항 수 (n) | 주기 | 의미 |
|-----------|------|------|
| n = 1 | 1095일 | 3년 전체 추세 |
| n = 3 | 365일 | **연간 사이클** ⭐ |
| n = 6 | 182.5일 | 반년 주기 |
| n = 30 | 36.5일 | 월별 변화 |
| n = 100 | 10.95일 | 날씨 패턴 |

## 🎯 3DGS와의 연결점

| 푸리에 급수 | 구면 조화 함수 (SH) |
|-------------|---------------------|
| 1D 신호 분해 | 3D 구면 위 함수 분해 |
| 사인/코사인 기저 | SH 기저 함수 |
| 계수 $a_n, b_n$ | SH 계수 (보통 9~16개) |
| 신호 근사 | **방향별 색상 표현** |

### 3DGS에서의 실용적 이점

1. **메모리 효율**: 무한한 방향 정보를 소수의 SH 계수로 압축
2. **계산 효율**: 렌더링 시 빠른 색상 계산
3. **부드러운 보간**: 방향 변화에 따른 자연스러운 색상 전환

---

## 📝 핵심 요약

```
복잡한 것 = 간단한 기저 함수들의 조합
```

이 아이디어가 1D(푸리에) → 2D(2D 푸리에) → 3D 구면(SH)으로 확장되며, 최종적으로 3D Gaussian Splatting에서 view-dependent color를 효율적으로 표현하는 데 사용됩니다.
