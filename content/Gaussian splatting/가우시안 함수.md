가우시안 스플래팅에서 가우시안 함수를 사용하는 이유
- 미분 가능하다
-> 파라미터에 대해서 미분(가중치) 값을 계산할 수 있습니다.
- 함수의 모양이 부드러워서, 결과가 아주 잘 나온다.
-> 이미지를 만들어야 하기 때문에, 가우시안 함수를 사용할 때 이미지가 부드럽게 잘 나오기 때문이다.
## 1. 가우시안 함수를 사용하는 이유

### 수학적/통계적 중요성
1. **중심극한정리 (Central Limit Theorem)**: 독립적인 많은 확률변수들의 합은 원래 분포와 상관없이 가우시안 분포로 수렴
2. **자연 현상**: 실제 세계의 많은 현상이 가우시안 분포를 따름
   - 인구 집단의 키 분포
   - 실험 측정 오차
   - 기체 분자의 속도 (맥스웰-볼츠만 분포)
   - 전자 신호의 노이즈

3. **미분에 대한 닫힘 성질**: 가우시안의 미분은 여전히 가우시안 (다항식과 곱해진 형태)
   ```
   d/dx(e^(-x²)) = -2x · e^(-x²)
   ```

### 확률 밀도 함수 형태
확률론에서 가우시안(정규) 분포:
```
G(x) = 1/(σ√(2π)) × e^(-(x-μ)²/(2σ²))
```
- μ: 평균 (기댓값) - 분포의 중심
- σ: 표준편차 - 퍼짐 정도
- σ²: 분산
- 1/(σ√(2π)): 정규화 상수 (전체 확률이 1이 되도록)

## 2. 1D 가우시안 함수의 대화형 시각화

노트북에서는 슬라이더를 통한 **대화형 시각화**를 제공:

### 주요 매개변수
- **μ (평균)**: 가우시안과 그 미분을 좌우로 이동
- **σ (표준편차)**: 
  - 작을 때: 좁고 급한 곡선, 가파른 미분
  - 클 때: 넓고 완만한 곡선, 부드러운 미분
- **n (범위)**: ±nσ 범위 표시 (경험적 법칙: n=1 ~68%, n=2 ~95%, n=3 ~99.7%)

### 핵심 관찰
- 미분은 μ에서 정확히 0을 통과 (가우시안의 최고점)
- 미분은 가우시안의 최대 곡률 지점에서 가장 가파름

## 3. 2D 가우시안 함수 (비등방성/타원형)

### 기본 공식 발전

**1차원 기본형:**
```
G(x₀) = e^(-½x₀²)
```

**2차원 기본형:**
```
G(x₀, y₀) = e^(-½(x₀² + y₀²))
G(x₀) = e^(-½x₀ᵀx₀)
```

### 변환 관계 (Transformation)

**순방향 변환** (정규화된 공간에서 세계 공간으로):
```
x = RS·x₀ + μ
```

**역방향 변환** (세계 공간에서 정규화된 공간으로):
```
x₀ = (RS)⁻¹(x - μ)
```

### 행렬 구성요소

**회전 행렬 R:**
```
R = [cos θ  -sin θ]
    [sin θ   cos θ]
```

**크기 행렬 S:**
```
S = [σₓ   0 ]
    [0   σᵧ]
```

### 최종 2D 가우시안 공식
```
G(x) = e^(-½(x-μ)ᵀΣ⁻¹(x-μ))
```
여기서 공분산 행렬: **Σ = RSS^TR^T**

## 4. 핵심 구현 함수

### `gaussian_2d_anisotropic` 함수
비등방성(회전 가능한 타원형) 2D 가우시안을 구현:

**단계별 구현:**
1. **이동**: (x - μ) 계산
2. **크기 행렬 생성**: S = diag(σₓ, σᵧ)
3. **회전 행렬 생성**: R(θ)
4. **공분산 행렬 계산**: Σ = RSS^TR^T
5. **역공분산 계산**: Σ⁻¹
6. **이차 형식 평가**: (x-μ)ᵀΣ⁻¹(x-μ)
7. **가우시안 값 계산**: e^(-½ × 이차형식)

### 대화형 2D 시각화 기능
- **3D 표면 보기**: 회전된 언덕 형태
- **2D 등고선 보기**: 타원형 등고선
- **실시간 매개변수 조정**:
  - μₓ, μᵧ: 중심 이동
  - σₓ, σᵧ: 각 방향 퍼짐 조정
  - 회전각: 가우시안 방향 조정

### 주요 통찰
- **공분산 Σ = RSS^TR^T**는 회전 시 비대각 항을 생성
- σₓ = σᵧ인 원형 가우시안은 회전 효과가 없음
- 변환 파이프라인: 이동 → 크기조정 → 회전

## 5. 2D 가우시안 스플랫 래스터화

### 래스터화 알고리즘
1. **픽셀 격자 생성**: 이미지를 2D 픽셀 배열로 정의
2. **각 픽셀에 대해**:
   - 이동 벡터 계산: (x - μ)
   - 공분산 행렬 구성: Σ = RSS^TR^T
   - 가우시안 평가: G = e^(-½(x-μ)ᵀΣ⁻¹(x-μ))
   - 색상과 곱하여 RGB 값 계산
3. **결과 저장**: 출력 이미지의 픽셀 색상/강도

### 효율적인 구현
- **einsum을 사용한 배치 계산**: 개별 픽셀 반복 대신
- **모든 픽셀 좌표를 단일 배열로 평면화**
- **한 번에 모든 가우시안 값 계산**
- **원래 이미지 차원으로 재구성**
