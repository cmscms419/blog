# 미분 가능한 렌더링 - 가우시안 스플랫팅 정리

## 1. 핵심 개념

### 2D 가우시안 함수와 공분산 행렬

2D 가우시안 스플랫은 **9개의 훈련 가능한 매개변수**로 정의됩니다:

1. **위치** μ = [μₓ, μᵧ]ᵀ - 2D 공간의 중심점 (2개 매개변수)
2. **크기** σ = [σₓ, σᵧ]ᵀ - 가로/세로 크기 (2개 매개변수) 
3. **회전각** θ - 방향각(라디안) (1개 매개변수)
4. **투명도** α ∈ [0,1] - 투명도/불투명도 (1개 매개변수)
5. **색상** rgb = [r,g,b]ᵀ - RGB 색상 벡터 (3개 매개변수)

### 가우시안 커널 함수

점 p에서의 가우시안 커널 값:

```
f(p) = exp(-1/2 × (p-μ)ᵀ × Σ⁻¹ × (p-μ))
```

여기서 Σ는 **공분산 행렬**로 가우시안의 모양과 방향을 결정합니다.

### 공분산 행렬 구성

회전 행렬 R(θ):
```
R(θ) = [cos θ  -sin θ]
       [sin θ   cos θ]
```

크기 행렬 S:
```
S = [σₓ   0 ]
    [0   σᵧ]
```

공분산 행렬:
```
Σ = R(θ) × S × Sᵀ × R(θ)ᵀ
```

### 공분산 역행렬 계산

효율적인 역행렬 계산:
```
Σ⁻¹ = R(θ) × [1/σₓ²   0   ] × R(θ)ᵀ
              [0     1/σᵧ²]
```

## 2. 렌더링 파이프라인

### 가산 혼합 렌더링

픽셀 p의 최종 색상은 모든 가우시안의 기여도를 합산:

```
I(p) = Σᵢ₌₁ᴺ αᵢ × rgbᵢ × fᵢ(p; μᵢ, Σᵢ)
```

**핵심 특징**:
- 각 가우시안이 독립적으로 회전 가능
- 투명도와 색상이 별도로 최적화
- 모든 곳에서 미분 가능 → 경사하강법에 최적

## 3. 수학적 핵심

**공분산 역행렬이 핵심입니다**:
- S_diag = [[1/σₓ², 0], [0, 1/σᵧ²]]는 (SSᵀ)⁻¹의 성분을 올바르게 나타냄
- 직접 역행렬 계산 방식이 수학적으로 정확함
- 모든 연산을 PyTorch 텐서로 수행하여 경사 보존
