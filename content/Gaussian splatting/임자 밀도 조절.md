## 📚 **핵심 개념 3가지**

### **1️⃣ Splitting (분할)**
- **목적**: 너무 커진 가우시안을 두 개로 나누기
- **조건**: `largest_sigma > max_sigma` (기본값: 0.35)
- **방법**: 
  - 주축(major axis) 방향으로 분할
  - 두 자식 가우시안으로 분리 (opacity는 절반으로)
  - 크기는 70%로 축소
- **핵심 코드**: `large_mask = largest > max_sigma`

### **2️⃣ Cloning (복제)**
- **목적**: gradient가 높은 곳에 가우시안 추가
- **조건**: `grad_norm > grad_threshold` & 크기가 작음
- **방법**:
  - 같은 위치에 동일한 가우시안 복사
  - 원본은 momentum 유지, 복제본은 momentum=0
  - 이후 iteration에서 자연스럽게 분리됨
- **핵심 코드**: `clone_mask = grad_norms > grad_threshold`

### **3️⃣ Pruning (제거)**
- **목적**: 투명해진 쓸모없는 가우시안 삭제
- **조건**: `opacity < min_opacity` (기본값: 0.01)
- **방법**: keep_mask로 필터링하여 제거
- **핵심 코드**: `keep_mask = opacities >= min_opacity`

## 🔧 **기술적 구현**

### **핵심 함수들**
1. `split_gaussians_if_large()` - 큰 가우시안 분할
2. `clone_gaussians_if_high_grad()` - 높은 gradient 가우시안 복제
3. `prune_gaussians()` - 투명한 가우시안 제거
4. `extend_optimizer_with_new_points()` - optimizer에 새 점 추가
5. `remove_from_optimizer()` - optimizer에서 점 제거

### **Optimizer State 관리**
- **추가 시**: 기존 파라미터 업데이트 + 새 파라미터 concatenate
- **제거 시**: keep_mask로 필터링
- **Adam momentum 보존**: exp_avg, exp_avg_sq 상태 유지

## 📊 **Loss Function: L1 + D-SSIM**

### **수식**
```
𝓛 = (1-λ)𝓛₁ + λ𝓛_D-SSIM
λ = 0.2 (기본값)
```

### **SSIM의 핵심 특징**
- **❌ 오해**: "밝기만 비교한다"
- **✅ 실제**: RGB 각 채널별로 독립 계산 → **색상 정보 포함**
- **3요소**: Luminance(밝기) + Contrast(대비) + Structure(구조)
- **11×11 윈도우**: 지역적 패턴 비교로 인간 시각과 유사
[[SSIM]]
### **조합의 이유**
- **L1 (80%)**: 픽셀 단위 색상 정확도
- **D-SSIM (20%)**: 구조적 품질과 지각적 유사성
- **결과**: 색상 정확 + 구조 선명한 고품질 이미지

## 💡 **핵심 인사이트**

### **1. 적응적 표현**
- 필요한 곳에 자동으로 더 많은 가우시안 배치
- 불필요한 영역에서는 가우시안 제거
- 동적 밀도 조절로 효율성 극대화

### **2. 회전의 중요성**
- 2D 회전 행렬로 타원 방향 조절
- 삼각형 가장자리와 같은 지향성 특징에 정렬
- 전체 2×2 공분산 행렬 표현으로 유연성 확보

### **3. Optimizer State 보존**
- 기존 가우시안의 학습된 momentum 유지
- 새 가우시안은 zero momentum으로 시작
- 이 차이로 인해 복제된 가우시안들이 자연스럽게 분리

```python
# Splitting 조건
large_mask = sigmas.max(dim=1).values > max_sigma

# Cloning 조건  
grad_norms = torch.norm(mus_grad, dim=-1)
clone_mask = grad_norms > grad_threshold

# Pruning 조건
keep_mask = opacities >= min_opacity

# Loss 계산
loss = 0.8 * L1_loss + 0.2 * D_SSIM_loss

# SSIM 공식
SSIM = (2μ₁μ₂ + C₁)(2σ₁₂ + C₂) / ((μ₁² + μ₂² + C₁)(σ₁² + σ₂² + C₂))
```
