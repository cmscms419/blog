# 형변환

# 1. static_cast

기본 클래스에 대한 포인터를 파생 클래스에 대한 포인터로 변환하는 등의 작업에 사용할 수 있다.

하지만 안전한 것은 아니다.

강제 타입 변환

```cpp
static_cast<new_type>(expression)
```

compile 시간에 형변환에 대한 타입 오류를 잡는다.

실수와 정수, 열거형과 정수형, 실수와 실수 사이의 변환 등을 허용한다.

포인터 타입을 다른것으로 변환 하는 것을 허용하지 않습니다. 그러나, 상속 관계에 있는 포인터 끼리 변환은 가능하다.

# 2. dynamic_cast

```cpp
dynamic_cast<type-id>(expression)
```

- 안전한 다운캐스팅으로 불린다.
- 부모 클래스의 포인터에서 자식 클래스의 포인터로 다운 캐스팅 해주는 연사자 입니다.
- 런타임 시간에 실제로 해당 타입이 다운 캐스팅이 가능한지 검사하기 때문에, 런타임 비용이 조금 높다.
- type-id이 포인터인 경우 expression의 형식은 포인터여야 하며, type-id이 참조인 경우에는 l 값이어야 합니다.

# 3. reinterpret_cast

```cpp
reinterpret_cast<type-id>(expression)
```

**2개의 서로 다른 형에 대해 완전히 새로운 재해석을 시도한다.**

- OS나 컴파일러에 의존적이거나, 안전하지 않은 형에 대해 형 변환을 한다.
- 포인터가 다른 포인터 형식으로 변환될 수 있도록 합니다. 또한 정수 계열 형식이 포인터 형식으로 변환될 수 있도록 하고 그 반대로도 변환될 수 있도록 합니다.

# 4. const_cast

```cpp
const_cast<T>(expr)
```

- cv-한정자(cv-qualified)로 선언된 변수의 const 및 volatile 성질을 버리기 위해 사용한다.

→ cv-qualified: const와 volatile을 의미한다.

1. volatile: 변경 가능하다.
2. const : 변경 X, 상수이기 때문에

# 6. 위험

```cpp
ip2 = (int *)ip;
```

char* → int*로 casting한 것이다.

```cpp
jp2 = (int *)jp;
```

const 성질을 버리기 위해 casting 한 것이다.

똑같은 형식을 사용했지만, 캐스팅 목적이 다르다. 

→ 이러면 **캐스팅의 의도를 파악하기가 어렵다.**

# CODE

```cpp
// main.cc
#include "./typecast.h"

void StaticCast_ex();
void DynamicCast_ex();
void ReinterpretCast();
void ConstCast();
void Casting();
void func(void *v);

int main() {
  StaticCast_ex();
  DynamicCast_ex();
  ReinterpretCast();
  ConstCast();
  Casting();

  return 0;
}

void func(void *v) {
  int i = reinterpret_cast<int>(v);
  cout << i << endl;
}

void StaticCast_ex() {
  A s_a;
  C s_c;
  B *s_b;

  // static_cast
  cout << "static_cast" << endl;
  cout << "static_cast DownCast" << endl;

  // static_cast DownCast
  s_b = static_cast<B *>(&s_a);
  s_b->Print();

  A *s_aa;
  B s_bb;

  // static_cast UpCast
  cout << endl;
  cout << "static_cast UpCast" << endl;

  s_aa = static_cast<A *>(&s_bb);
  s_aa->Print();
}

void DynamicCast_ex() {
  // Dynamic_cast
  cout << endl;
  cout << "dynamic_cast" << endl;

  // Dynamic_cast DownCast
  cout << "Dynamic_cast DownCast" << endl;
  D_A *d_a = new D_A();
  d_a->Show();

  D_B *d_b = dynamic_cast<D_B *>(d_a);
  if (d_b == nullptr) {
    cout << "Runtime Error" << endl;
  } else {
    d_b->Show();
  }

  // Dynamic_cast DownCast 2
  cout << endl;
  cout << "Dynamic_cast DownCast 2" << endl;
  D_A *d_aa = new D_B();
  d_aa->Show();

  D_B *d_bb = dynamic_cast<D_B *>(d_aa);
  if (d_bb == nullptr) {
    cout << "Runtime Error" << endl;
  } else {
    d_bb->Show();
  }

  delete d_a;
  delete d_aa;
}

void ReinterpretCast() {
  // reinterpret_cast
  cout << endl;
  cout << "reinterpret_cast" << endl;

  func(reinterpret_cast<void *>(5));

  typedef void (*PFV)();

  PFV pfunc = reinterpret_cast<PFV>(func);

  pfunc();
}

void ConstCast() {
  // const_cast
  cout << endl;
  cout << "const_cast" << endl;

  int i = 100;
  const int *ip = &i;
  cout << *ip << endl;

  int *ip2 = const_cast<int *>(ip);
  *ip2 = 200;
  cout << *ip << endl;
}

void Casting() {
  // C casting
  cout << endl;
  cout << "C casting" << endl;
  char i = 1;
  char *ip = &i;
  int j = 100;
  const int *jp = &j;

  int *ip2;
  int *jp2;

  ip2 = (int *)ip;
  cout << *ip2 << endl;
  jp2 = (int *)jp;
  cout << *jp2 << endl;
}
```

```cpp
// typecast.h

#ifndef TYPECAST_TYPECAST_H_
#define TYPECAST_TYPECAST_H_

#include <iostream>

using std::cout;
using std::endl;

class A {
 public:
  A() { a = 1; }
  void Print() { cout << "class A : " << a << endl; }

 private:
  int a;
};

class B : public A {
 public:
  B() { b = 2; }
  void Print() { cout << "class B : " << b << endl; }

 private:
  int b;
};

class C : public B {
 public:
  C() { c = 3; }
  void Print() { cout << "class C : " << c << endl; }

 private:
  int c;
};

class D_A {
 public:
  D_A() { cout << "D_A()\n"; };
  virtual ~D_A() { cout << "~D_A()\n"; };

  void Show() { cout << "This is D_A Class\n"; }
};

class D_B : public D_A {
 public:
  D_B() { cout << "D_B()\n"; };
  virtual ~D_B() { cout << "~D_B()\n"; };

  void Show() { cout << "This is D_B Class\n"; }
};

#endif  // !TYPECAST_TYPECAST_H_
```
