# 순수 가상 클래스

class를 왜 사용하는가?

→ 이 질문은 구조체를 왜 사용하는가?

→ 구조체는 data 집합을 사용하기 위해서

→ class는 data 집합과 data 관리를 위한 함수를 사용하기 위해서

class의 문제

상속

- 불필요한 기능 추가
- 다이아몬드 상속 → error
- 상속이 많으면, 자식의 크기가 많아진다.
    - 포함을 권장

virtual이 나온 이유

기준을 삼기 위해서

| 정적 선언 | 동적 선언 |
| --- | --- |
| 자식에서 부모의 클래스가 생성됩니다. | virtual table이 생성됩니다 |
|  | virtual은 인스턴스화 안됩니다. |

클래스의 형변환은 주소가 변환됩니다.
