# 포인터란 무엇인가

1. 컴퓨터에서 데이터를 어떻게 저장하는지
    1. 데이터를 0과 1로 저장을 한다. 하지만 이것만을 가지고는 어떤 데이터인지 알 수 없기 때문에 데이터 타입이 어떻게 되어있는지 알아야한다.
        1. 실수형
        2. 정수형
        3. 참/거짓형 등등
    2. 데이터 타입이 몇 byte까지 쓰는지 에 따라서 데이터 크기와 종류가 다르게 저장된다.
2. 메모리란 무엇인지
    1. 컴퓨터에서 정보를 처리하기 위해 일시적으로 정보를 보관하는 기억장치
    2. 주로  RAM을 의미한다.
3. 메모리에 데이터를 어떻게 저장하는지
    1. 
        
        ![[/RAM1.png]]
        
    2.  메모리는 하나의 긴 줄 이라고 보고, 일정한 칸으로 나누어져 각 칸에 데이터를 저장할 수 있습니다.
    3. 
        
        ![[/RAM2.png]]
        
    4. 임의의 공간에 저장되어진다. 원하는 데이터의 값을 찾기위해 순차적으로 찾는 것이 아닌, 한번에 원하는 데이터를 찾을 수 있다.
    5. 예를 들어 변수 x,y에 90과 True 값을 저장하면 실제 메모리 어딘가에 90과 true가 저장됩니다.
4. 메모리 주소와 주소 내 데이터의 관계 
    1. 메모리에 데이터를 저장하면, 임의의 공간에 저장됩니다. 하지만 메모리에 데이터가 너무 많으면 원하는 데이터를 찾는 것이 어렵습니다. 이 문제를 해결해 주는 것이 바로 메모리 주소입니다. 각 데이터 마다 메모리 위치를 식별하는 주소를 만들어서, 그 주소를 통해서 빠르게 원하는 데이터를 찾을 수 있게 도와줍니다.
5. 메모리에 접근하는 법 
    
    ```cpp
    int *p
    int x = 13
    
    p = &x
    ```
    
    1. 
    
    ![Untitled](attachments/Untitled_17.png)
    
    p는 int형 포인터라서, int 형 데이터의 주소값을 받을 수 있다. 그 주소값을 이용해서 메모리에 접근 할 수 있다.
    
6. 메모리에 있는 데이터를 접근하는 법 
    
    ```cpp
    int *p
    int x = 13
    
    p = &x
    ```
    
    1. p*이 가지고 있는 X의 메모리 주소를 찾아가서, 메모리 주소에 있는 데이터를 접근할 수 있습니다.
7. 배열과 메모리 주소 관계 
    1. 배열의 이름은 배열의 시작 주소를 가리킵니다. 이 의미는 배열의 이름 자체가 바로 메모리 주소를 가리킨다는 것입니다.

    주소 연산으로 구할 수 있는 이 주소를 저장하는 포인터를 2중 포인터 이다.
