# 무지형 (Void)

Void Type

- Void는 메모리를 차지하지 않아서, 자료형으로 변수 선언은 안 된다.
- 포인터 변수 선언은 가능하다.
- 보이드 포인터로 형변환을 할 수 있다.

## 리터럴 상수

c++ 14 바이너리 상수

변하지 않는 숫자

const를 선언하면, 값을 바꿀 수 없다.

숫자 뒤에 자료형의 형태를 알려줍니다.

```cpp
int x = 12 -> 10진수
int x = 012 -> 8진수
int x = 0xF -> 16진수
```

16진수로 쓰는 경우가 많다.

## 심볼릭 상수

C++11 constexpr

const 값을 고정시킨다.

- 반드시 초기화 시켜줘야한다.
- 파라미터의 값을 바꾸는 경우는 거의 없다.
    - 값을 복사해서, 다른 변수를 통해서 바꾼다.
- constexpr → const expression
    - 컴파일 타임의 값이 완전히 결정되는 상수라는 것을 컴파일하면서 체크하겠다는 의미

매크로를 상수를 대체하기 위해서 사용하는 경우는 드물다.

이유 : 

디버깅 힘듬

해당 매크로가 적용되는 범위가 너무 넓다 → 다른 값으로 사용하고 싶을 때, 변경하지 못한다.
