# 전역 변수, 정적 변수, 내부 연결, 외부 연결

전역 변수 : 어디서나 사용가능한 변수 → 최대한 적게 사용해야 한다.

전역 연산자 : 다른 영역에 사용된 값을 사용할 수 있다. (::)

정적 변수 :  static 변수

→ static : 정적

→ 변수의 메모리가 정적으로 선언이 된다.

→ 보통 범위를 넘어가면 사라진다.

→ 해당 범위 안에서 초기화가 1번만 일어난다. 그리고 해당 변수를 재사용한다. 처음에 정적 변수를 선언 할 때는 초기화를 반드시 해야한다.

→ static 변수는 존재하고 있다. 하지만 다른 영역에서는 접근하기 어렵다. 포인터를 사용하면 복잡하다. 차라리 객체지향을 사용하는 것이 좋다.

extern을 사용하면, 다른 파일에 있는 함수, 변수를 사용할 수 있다.
